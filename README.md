## **Start Locally**

1\. Clone git repository:

```bash
git clone https://github.com/RedberryInternship/kakha-chankseliani-covid-questionnaire.git
```

2\. Install dependencies

```bash
npm install
```

3\. Run in development

```bash
npm run dev
```
